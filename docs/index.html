[CLAUDY]

AI Coding Assistant with Pixel-Art Visualization

Watch Claude Explore Your Codebase

Claudy wraps Claude CLI in a beautiful dual-view interface. Use the terminal like normal, or switch to the RPG view and watch Claude's character navigate a pixel-art world as it reads files, runs commands, and writes code.

✦ Features

💻

Full Terminal

Complete xterm.js terminal with all Claude CLI features. Search, scroll, copy, and paste just like your normal terminal.

🎮

RPG Visualization

Watch Claude's character move through a pixel-art world. Tool usage drives character animations and movement.

💬

Chat Panel

Clean chat interface showing Claude's responses. Type messages that go directly to the same PTY session.

⚡

Real-time Sync

Terminal and RPG views share a single PTY. Everything stays perfectly synchronized.

System Requirements

macOS 11+ • Apple Silicon (M1/M2/M3) • Claude CLI installed
